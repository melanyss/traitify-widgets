# Traitify Widgets

## Initializing The Library
```
Traitify.setHost('your host url')
        .setPublicKey('your public key');

Traitify.ui.assessmentId('an assessment id you have generated via a server side client')
           .render();
```

## Setup

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```
